# Hermetic settlement tests

These run the **real TxLINE oracle binary in-process** (via [LiteSVM](https://github.com/LiteSVM/litesvm))
against its **real anchored `daily_scores_roots`** and a **real finalised proof** —
no devnet, no validator, no mock oracle. Our `settle()` CPIs the genuine
`validate_stat`, and the tests assert the outcome is *derived* from its verdict and
that every fraud path reverts.

Why it matters: a settlement market can claim anything in a README. This proves,
offline and deterministically, that the outcome comes from TxLINE's cryptographic
verdict — and that tampering with the proof, the fixture, the roots account, the
finality period, or the market's question all revert.

## Run

```bash
npm install
npm run dump    # once — extracts artifacts/ (txline.so, fulltime.so, roots.json, proof.json) via devnet RPC + TxLINE API
                #        (needs CREDS=~/fulltime-keys/txline-creds.json). Artifacts are committed, so you can skip this.
npm test        # node --test settle.test.mjs
```

## What's covered (16 tests, all green)

- **Happy path** — settle CPIs the real `validate_stat`, reads the verdict, and the market resolves **YES** for a real 1–0 (derived, not caller-supplied).
- **Attack matrix (all revert):** tampered goal value (Merkle fails inside `validate_stat`), tampered team-2 goal, tampered sub-tree proof node, tampered summary root, tampered stat Merkle proof, wrong fixture (`FixtureMismatch`), substituted roots account (`RootsMismatch`), non-final period (`NotFinal`), predicate not matching the market question (`PredicateMismatch`), settling a Settled market (`NotOpen`), truncated args (`InstructionDidNotDeserialize`).
- **Invariants:** `create_market` initializes Open with zero pools; the same fixture can't be created twice; two independent settlements of the same proof are deterministic.

`artifacts/` is regenerated by `dump-artifacts.mjs` — it pulls the TxLINE and
Fulltime program ELFs out of their on-chain ProgramData accounts over plain
JSON-RPC (no `solana` CLI needed), plus the real roots PDA and a real
period-100 proof.
